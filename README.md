CMAR Report Portal
==================
##codename: CR

<br>
###What is CR?
A cut and paste CMS application.

###How do I use it?
 run the cmar.py file with your
 python interpreter and the application will
 greet you on http://localhost:5000/
    
###Is it tested?
  You betcha.  Run the `cmartests.py` file to see
 the tests pass.
